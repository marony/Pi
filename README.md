# モンテカルロ法(ランダムでシミュレーション)で円周率を求める

　1辺の長さがAの正方形の中に直径Aの円を置き、その正方形の中にランダムに点を打ちます。
　全体の試行回数とその点が円の中に入った回数を数え、円の中に入った回数 * 4 / 試行回数が円周率に収束していきます。
